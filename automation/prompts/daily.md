## 지시사항

위 종목의 일일 moat 분석을 수행하라.

1. 웹 검색으로 최근 24시간 내 해당 종목 관련 뉴스, 공시, 애널리스트 코멘트를 확인하라.
2. 현재 moat thesis 대비 변동 사항을 평가하라. 버핏식 4축 기준: pricing power, switching cost, network effect, cost advantage.
3. 아래 출력 형식을 정확히 따라 응답하라.

## 출력 규칙

- **호재/악재 개수 제한**: 각 섹션 최대 3개. 우선순위 높은 순서로. 4개째는 출력 금지.
- **정량 강제**: 각 bullet에 숫자/%/금액 포함 필수. 예: "Q1 SSSG +3.8%", "EPS $5.11 (+81% YoY)". 정량 데이터 없는 정성적 코멘트는 금지.
- **쉬운 표현**: 전문용어 최소화. 약자 사용 시 한 번은 풀어쓰기. 예: "ROIC(투하자본수익률)".

## 출력 형식

### Moat 상태
{Strong|Weakening|Uncertain} ({유지|변동})

### 호재
• (moat 강화 신호. 최대 3개. 정량 데이터 필수. 출처 표시 금지. 없으면 "없음")

### 악재
• (moat 약화 신호. 최대 3개. 정량 데이터 필수. 출처 표시 금지. 없으면 "없음")

### Valuation
• 현재가: $XXX
• (FCF Yield, 배당수익률, PER 등 핵심 지표 1~3개. 출처 표시 금지)

### 종합평가
한 문장으로 종합 판단. 출처/소스/뉴스명 포함 금지. 팩트와 판단만.

## 절대 금지

- 출력 끝에 `Sources:` / `References:` / `출처:` / `참고문헌` 등 어떤 형태로도 출처/링크/매체 목록 섹션을 부착하지 말 것.
- URL, 매체명, 정확한 헤드라인 인용 일체 금지 (각 섹션 내부 + 트레일링 모두).
- 사실의 *내용*만 기술. "Bloomberg 보도", "CNBC에 따르면" 같은 출처 지칭 표현도 금지.
