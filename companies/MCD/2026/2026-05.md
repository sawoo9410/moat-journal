# MCD — 2026년 5월

## 2026-05-07

### Moat 상태
Strong (유지)

### 호재
• Q1 2026 어닝 비트 — EPS $2.83 (컨센 $2.75), 매출 +9% $6.52B (컨센 $6.49B)
• 글로벌 SSSG +3.8% (US +3.9%, IOM +3.9%) — 컨센 3.7% 상회, 52주 저점 분위기 반전
• 프리마켓 +3%↑ — value 메뉴 ("3 for 3", McValue 2.0) + Netflix 콜라보의 affordability 대응 전략 효과 확인
• CEO: GLP-1이 "아직 물질적 영향 없음", 프로틴 카테고리 수혜 가능성 언급 — castle 외부 위협의 즉각 침식은 미발현

### 악재
• US SSSG가 "customers spending more when they visited"에 의존 — **트래픽 회복은 아직 미확인**, 객단가 위주 성장
• value 메뉴 의존도 상승 = thesis의 "가격결정력 한계 신호" 재확인 (가격 인상으로는 더 이상 SSSG를 못 만든다)
• 어닝 직전 주가 52주 저점까지 하락 — 시장은 이미 모멘텀 둔화를 가격에 반영 중
• GLP-1: CEO 본인이 "소비자 행동 변화" 인정 — 점진적 침식 가설 살아있음

### Valuation
• 현재가: ~$285 부근 추정 (52주 저점 직후 어닝 비트 반등, 정확한 종가 미확인)
• UBS 목표가 $365 (≈ +28% upside), Wall St 컨센 중앙값 $354 — 20 Buy / 15 Hold / 2 Sell
• 배당수익률: dividend aristocrat 지위 유지, 52주 저점 부근에서 ~2.5% 내외 추정 (정확치 미확인)

### 한줄평
어닝 비트 + SSSG 회복으로 단기 thesis 침식 우려는 진정됐지만, 성장 동력이 "객단가 + value 메뉴"로 이동한 구조는 가격결정력 한계라는 thesis를 강화 — moat은 Strong holding이지 widening은 아니다.
